# EDA-on-Census-India-2011-Language-Dataset

Download all files in a folder named as '21111045-assign2'. Run assign2.sh after making working directory to folder - '21111045-assign2' to run the entire assignment by command -> sh assign2.sh

Runtime = 25 seconds

Python packages required :

- python : To execute all codes.
- csv	:  to manage data
- pandas : for dataset handling
- numpy : support for calculations
- math :  for mathematical operations
- scipy.stats  : for finding p values in Question 2,3
- warnings : to ignore unnecessary warnings
- xlrd: For dealing with some older version of the excel files (.xls)
